Java Script Game

Kitty Crunch! is a small browser-based game developed as a group project, where we collaborated on concept, logic, and implementation. The goal of the game is to collect falling tacos while avoiding bombs, with the difficulty increasing over time.

The game is built using HTML, CSS, and vanilla JavaScript, with JavaScript handling all core gameplay logic. We used JavaScript to control player movement through keyboard input, dynamically spawn game objects, and manage collision detection between the player, tacos, and bombs. Object positions are updated continuously using bounding box calculations to create smooth movement and responsive interactions.

Game state such as score, high score, and difficulty progression is managed through JavaScript variables and timers. The spawn rate increases gradually to make the game more challenging, while the high score is stored using localStorage so it persists between sessions.

Working as a group allowed us to divide responsibilities, test ideas quickly, and refine both gameplay and user experience through iteration. This project helped strengthen my understanding of JavaScript fundamentals, DOM manipulation, event handling, and collaborative development.
