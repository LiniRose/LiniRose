Form Validation

This project is a browser-based character creation form inspired by Dungeons & Dragons, developed using HTML, CSS, and JavaScript. The goal was to create an interactive and user-friendly form that dynamically responds to user input while demonstrating real form validation and frontend logic.

The application uses JavaScript to manage form validation, update UI elements in real time, and enhance the user experience. Different sections of the form (such as race, class, and background) dynamically display related information by fetching structured data from external JSON files. This allows the form to feel responsive and data-driven rather than static.

I also implemented custom validation logic for elements such as password confirmation, ensuring input consistency before submission. Additional features include image upload with live preview, range-based input for character height with real-time formatting, and conditional navigation once the form is correctly completed.

Overall, the project helped strengthen my understanding of DOM manipulation, event handling, form validation, and working with external data sources in JavaScript, while also reinforcing the importance of creating intuitive user experiences.
